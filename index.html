How to Install/Get Pixelmon!

Today, I'll be showing you how to download/install Pixelmon MinecraftMod

Where to find the Mod and Forge

First off, you will need to go to the Pixelmon Mod website. Link-http://www.pixelmonmod.com/downloads.php-Copy and past please.

Then depending on what server you play you will need a different version of pixelmon. For the server I play you will need the "3.5.1-1.7.10" Version so click that Then you will need to skip the ad and download the Mod.You will need to go to your .minecraft Folder then make a new file called "mods" And move the Mod there from your downloads folder.

Now for forge. As in the Pixelmon Mod you will need a different version of forge for different versions of Pixelmon. For example, the "3.5.1" Pixelmon needs the 1.7.10 forge. So you will need to go to the "Minecraft Forge" website. Here is the link "http://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.7.10.html" Copy and paste, it should look like this

You will now need to Pick the correct version of forge,and then download it and press "run".

How to use the mod/forge.

You will need to open up the 1.7.10 forge at least once. Then use the Mod.

Have fun in Pixelmon :D
